# SoftWindBears

Adds **Grizzly Bears** and **Panda Bears** to Minecraft (NeoForge 1.21.1).  
Originally made for the **SoftWind** modpack, but can be used standalone.

## Features
- **Grizzly Bear** — spawns in warm taiga biomes
- **Panda Bear** — spawns in jungle biomes (replacing vanilla pandas)
- **Bear Skin** item dropped by grizzly bears (0–2, affected by Looting)
- Spawn eggs for both bears

## Config

File: `config/softwindbears-common.toml`

Available options:
- `areFriends`
- `bearHealth`
- `bearFollowRange`
- `bearAttackDamage`
- `bearSpeed`
- `bearAngerMinDuration`
- `bearAngerMaxDuration`

## Spawning
To change spawn biomes, weights, or group sizes, override the mod's biome modifier files with your own datapack.
